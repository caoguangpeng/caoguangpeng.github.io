Vim常用命令

发表于 2015-07-13   |   分类于 其他   |  

Vim有三种模式：命令模式(Command mode)，插入模式(Insert mode)和底线命令模式(Last line mode)。
用户刚刚启动Vim，便进入了命令模式。

i 切换到插入模式，以输入字符。
: 切换到底线命令模式，以在最底一行输入命令。


命令模式

光标移动

命令 作用
w, e, W, E 跳到单词的后面
b, B 跳到单词的前面
^ 到行首
$ 到行尾
gg 到第一行
G 到最后一行
[N]G 到指定行


编辑

命令 作用
J 合并下一行到上一行
u 撤销上一步操作
ctrl+r 恢复上一步操作
[N]>> 一行或N行往右移动一个tab
[N]<< 一行或N行往左移动一个tab


搜索

命令 作用
/pattern 搜索
?pattern 反方向搜索
n 到搜索结果的后一个目标
N 到搜索结果的前一个目标


选中

命令 作用
v 选中一个或多个字符（配合方向键）
V 选中一行


剪切和复制

命令 作用
dd 删除一行
dw 删除一个单词
[N]yy 复制一行或者N行
p 粘贴


进入到插入模式

命令 作用
i 插入到光标前面
o, O 新开一行
Esc 退出插入模式



底线命令模式

显示行号

set number

关闭

命令 作用
wq, x 保存并关闭
q 关闭（当前无改动前提下）
q! 强制关闭（放弃未保存的）
